NAME
       scrollspeed  -  set  the scroll wheel speed on a mouse using the System
       Preferences contol panel


SYNOPSIS
       scrollspeed [ N ]


DESCRIPTION
       scrollspeed changes the mouse scroll wheel  speed  via  System  Prefer-
       ences.   Usage:   scrollspeed  N  [where N ranges from 0 (slowest) to 7
       (fastest)]
