NAME
       fdc -  change the frontmost finder window to the terminal pwd.

SYNOPSIS
       fdc [directoryname]

DESCRIPTION
       fdc changes the finder window to match the pwd of the terminal in which
       the command is issued, or to the  directoryname  if  that  argument  is
       specified.   fdc  when  issued without the optional directoryname argu-
       ment, is essentially the reciprocal operation of cdf.  The command  fdc
       is an osascript embedded in a shell script that updates the Finder.
