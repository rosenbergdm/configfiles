NAME
       gnotify -  invokes growlnotify

SYNOPSIS
       gnotify

DESCRIPTION
       gnotify [command] is a pre-command function that notifies the user when
       the command (provided as an argument) completes.  It  attempts  to  use
       growlnotify  if  present, otherwise it prints a message. If the command
       returns anything other than  0  (i.e.,  it  fails),  the  growl  window
       remains sticky.

       A  command  completed  that  returns 0 will terminate with a non-sticky
       growlnotify window. Use the command snotify if you want that one to  be
       sticky as well.


EXAMPLE
       gnotify date
