NAME
       a2preview

SYNOPSIS
       a2preview ascii-text-file

DESCRIPTION
       a2preview  is  a  wrapper for a2ps that makes an ascii text file into a
       pretty pdf. You have to install a2ps (for example with fink).

AUTHOR
        William Scott

SEE ALSO
       a2ps(1)
