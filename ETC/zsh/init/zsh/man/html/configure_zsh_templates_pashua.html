NAME
       configure_zsh_templates_pashua -  invoked by configure_zsh_templates

SYNOPSIS
       configure_zsh_templates_pashua

DESCRIPTION
       configure_zsh_templates_pashua  is  invoked  by  the configure_zsh_tem-
       plates function. It invokes the Pashua GUI version  useful  for  larger
       displays. This is the default.
