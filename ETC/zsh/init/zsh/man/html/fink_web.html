NAME
       fink_web - search the fink package website database


SYNOPSIS
       fink_web [packagename]

       fink_web -s PATTERN


DESCRIPTION
       fink_web  is  a function to search fink's package website database.  It
       takes a fink packagename as an optional argument,  eg:   fink_web  zsh-
       templates.  If the -s option is invoked, a search is performed to match
       PATTERN
