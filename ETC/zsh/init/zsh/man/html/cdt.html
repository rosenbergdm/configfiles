NAME

cdt  

SYNOPSIS

cdt [-f,-F]  

DESCRIPTION

cdt change the current working directory to that corresponding to the file displayed in TextMate's frontmost window

cdt -F change this and the Finder (-F)

cdt -f change the finder only (-f)

This is only useful if you are using the text editing program TextMate.app.
